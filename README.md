# Score Tools Web

三麻・四麻スコア計算、ポーカースコア精算、清一色「何切る」練習をブラウザで動かす静的Webアプリです。

## トップ構成

- Critical Sonic Sanma Score：三麻・四麻スコア計算
- Hadou Kaiwai Score：Poker Score / Transfer
- Chinitsu Quiz：清一色 何切る / 待ち当て

## 使い方

1. `index.html` をブラウザで開く
2. トップから「Critical Sonic Sanma Score」「Hadou Kaiwai Score」「Chinitsu Quiz」を選ぶ

## スコア計算ルール

### 三人麻雀
- 50000点持ち50000点返し
- ウマ：1位 +30 / 2位 0 / 3位 -30
- チップ：1枚5P

### 四人麻雀
- 25000点持ち30000点返し
- ウマ：1位 +30 / 2位 +10 / 3位 -10 / 4位 -30
- オカ：トップ +20
- チップ：1枚5P

## 保存

対局履歴と集計状態はブラウザの `localStorage` に保存されます。

## 友人に共有する方法

このフォルダをNetlify / GitHub Pages / Vercelなどに置くとURLで共有できます。

## 履歴表示

三麻/四麻の履歴は選択中のルールごとに分離表示されます。保存データ自体は同じブラウザ内に保持されます。

## 何切るクイズの正解判定

何切るクイズは、まずシャンテン数を優先し、同じシャンテン数の候補の中で受け入れ枚数が最大の打牌を正解にします。

## メンバー候補

名前入力欄の候補表示に加えて、各プレイヤー行の「選択」ボタンから登録メンバーを選べます。iPhone Safariでも3人目・4人目の候補選択が安定します。

## メンバー選択UI

各プレイヤー行の「選択」ボタンを押すと、登録済みメンバーがその場に一覧表示され、タップで選択できます。番号入力は不要です。

## チップ入力修正

三麻/四麻どちらでも、残り1人のチップ自動補完が安定するように修正しました。チップ欄はマイナス入力に対応し、途中入力の `-` を不正な数値として扱わないようにしています。

## チップ未入力の扱い

チップ欄は未入力なら0枚として扱います。全員チップなしの場合は空欄のままで保存できます。一部だけチップを入力した場合は、合計が0枚になるように入力してください。

## チップ入力仕様

チップは、三麻なら2人分、四麻なら3人分を入力すると、残り1人分を合計0枚になるよう自動補完します。空欄が2人以上ある場合は未入力扱いです。


## Hadou Kaiwai Score

- 表記は In / Out / Net / Transfer
- 単位は pt
- 円・金額・投資額・回収額などの表記は使わず、スコアとして扱います
- Net Check が 0 pt なら正常です
- Transfer は Net に基づいて最小に近い形で自動生成します
- 履歴はブラウザの `localStorage` に保存されます
